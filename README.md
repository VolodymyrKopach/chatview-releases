# ChatView — канал авто-оновлення

Публічний канал роздачі оновлень для десктоп-апки **ChatView**.
Апка при старті тихо читає `version.json`; якщо версія новіша за встановлену —
качає `payload.zip`, перевіряє sha256 і застосовує на наступному запуску.

- `version.json` — {version, url, sha256}
- `payload.zip` — OS-незалежна начинка (код + UI), одна на Mac і Windows

Тут лежать ТІЛЬКИ артефакти збірки. Жодних приватних даних.
